git 命令

git的常用命令

git help <command> 显示command的帮助
git show $id 显示某次提交的内容
git co -- <file> 抛弃工作区file文件的修改
git co . 抛弃工作区的修改
git add <file> 将工作file文件修改提交到本地暂存区
git add . 将所有工作文件修改提交到本地暂存区
git rm <file> 从版本库中删除file文件
git rm <file> --cached 从版本库中删除file文件，但不删除file文件
git reset <file> 从暂存区恢复到工作文件
git reset -- . 从暂存区恢复到工作文件
git reset --hard 恢复最近一次提交过的状态，即放弃上次提交后的所有本次修改

将git add, git rm和git ci等操作都合并在一起做
